## Social-Media Application Using:
1. ReactJs
2. NodeJs
3. MongoDB (To store userDetail)
4. Firebase (To store images/files)

## How to run Locally:
1. `Social-Application/client`
2. Run `npm install` to install all packages
3. Run `npm start` to run the server
